# 🐛 GitHub Issues Tracker

> **Track, manage, and organize your GitHub issues — all in one clean dashboard.**

GitHub Issues Tracker is a clean, responsive web app that lets you sign in and manage project issues fetched live from an API. Filter by status, search by keyword, and inspect each issue in detail — all from a minimal, distraction-free interface.

🌐 **Live Site:** [https://naimxofficial.github.io/GitHub-Issues-Tracker/](https://naimxofficial.github.io/GitHub-Issues-Tracker/)

---

## ✨ Features

| Feature | Details |
|--------|---------|
| 🔐 **Login System** | Username & password authentication with demo credentials |
| 📋 **Issues Dashboard** | View all project issues fetched live from an API in a responsive grid |
| 🟢 **Open / 🟣 Closed Status** | Issues are color-coded by status — green for open, purple for closed |
| 🔴 **Priority Badges** | Each issue shows its priority — High, Medium, or Low |
| 🏷️ **Labels** | Issues display tags like Bug, Help Wanted, and more |
| 🔍 **Live Search** | Search issues in real-time by keyword via API |
| 🗂️ **Filter by Status** | Filter issues by All, Open, or Closed with one click |
| 🪟 **Issue Detail Modal** | Click any issue card to open a full detail popup with description, assignee, priority & labels |
| ⏳ **Loading Spinner** | Smooth loading indicator while fetching data |
| 📱 **Fully Responsive** | Works seamlessly on mobile, tablet, and desktop |
| 🎨 **Clean UI/UX** | Minimal, readable design with clear visual hierarchy |
