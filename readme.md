# What is the difference between var, let, and const?
### var: var is function scoped. It's reassignable like let. In present days, it's not recommended to use.
### let: let is block scoped. It's reassignable. It can be accessed after declaration.
### const: const is also block scoped and can be accessed after declaration. But const cannot be reassigned.

# What is the spread operator (...)?
### Spread operator (...)  is an ES6 feature that allow us to expand an array or an object into individual elements.

# What is the difference between map(), filter(), and forEach()?
### map() creates a new array by applying a function to every element of the oiginal.
### filter() creates a new array containing only elements that pass a test.
### forEach() doesn't return anything. It only executes code for every item that is in an array.
